# StudyRecord Application

## プロジェクトの目的

このアプリケーションは、主にSpring Securityの学習と実践を目的として開発しました。ユーザー認証、認可、JWT（JSON Web Token）を用いたAPIセキュリティの実装を通じて、Spring Securityの基本的な概念と設定方法を習得することに重点を置いています。

## 主な機能

-   **ユーザー認証**: JWT (JSON Web Token) を利用したセキュアなログイン機能
-   **記事管理**: 学習記録として記事の投稿、閲覧

## 使用技術

-   **バックエンド**:
    -   Java 11
    -   Spring Boot 2.4.1
    -   Spring Security (JWT認証)
    -   MyBatis
-   **データベース**:
    -   PostgreSQL
-   **ビルドツール**:
    -   Apache Maven
-   **その他**:
    -   Lombok

## 必要環境

-   Java 11
-   Apache Maven
-   PostgreSQL
